	1. Build Image: docker build -t ubuntu_b_net:v2 -f net_dockerfile_v2 .
	2. Tag Image for ACR: docker tag ubuntu_b_net:v2 cnk8sstudies-guf7d4fafabxehdy.azurecr.io/nwapps/ubuntu_b_net:v2
	3. Login to Azure: az login
	4. Login to Container Registry: az acr login --name cnk8sstudies
	5. Push Tagged Image to ACR: docker push cnk8sstudies-guf7d4fafabxehdy.azurecr.io/nwapps/ubuntu_b_net:v2
	6. Verify by Listing Repos in ACR (v1): az acr repository list --name cnk8sstudies
	7. Verify by Listing Repos in ACR (v2): az acr repository list --name cnk8sstudies --output tsv
	8. Verify by Listing Tags in Specific Repo: az acr repository show-tags --name cnk8sstudies --repository nwapps/ubuntu_b_net  --output table
	9. CREATING CONTAINER FROM CLI
		a. Create a managed identity: portal search "managed identities" > Create...
		b. Assign the Acrpull role to the managed identity for the ACR (container registry), cnk8sstudies:  search+select "Container registries" > reg's IAM > Add…
		c. Get resource ID of the managed identity:
			"az identity show --resource-group csrg --name aci-container-acrpull --query id -o tsv"
				Note: aci-container-acrpull is the user-created managed identity.
		d. Create container-group with this command form:
			"az container create --resource-group csrg --name ubuntu-b-net-2b --image cnk8sstudies-guf7d4fafabxehdy.azurecr.io/nwapps/ubuntu_b_net:v2 --assign-identity <IDENTITY_RESOURCE_ID> --acr-identity <IDENTITY_RESOURCE_ID> --ip-address Public --os-type Linux"
				Note 1: actual command contains private info and is in my 242 OneNote notes
				Note 2: A running container instance is created (do you want to stop it?)
		e. Check container-group status:
			BRIEF: "az container show --resource-group csrg --name ubuntu-b-net-2b --query instanceView.state --output table"
			DETAILED: "az container show --resource-group csrg --name ubuntu-b-net-2 --output table"
		f. Stop the Container-Group:
			"az container stop --resource-group csrg --name ubuntu-b-net-2b"
	10. Start the Container-Group: az container start --resource-group csrg --name ubuntu-b-net-2b
	10. Check Status of Container-Group (brief): az container show --resource-group csrg --name ubuntu-b-net-2 --query instanceView.state --output table
